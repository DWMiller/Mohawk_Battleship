Detailed Description

The Controller class wraps around an IBattleshipController object and provides various utilities that are used by a Competition object.

This class is almost exclusively used by the Competition class.

There are three constants available that should be used when determining the side the Controller is on: None, Red, Blue. Use None as an indicator for no controller.
